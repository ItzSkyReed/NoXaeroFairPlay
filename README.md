# NoXaeroFairPlay

**Version:** 1.21.11+

**Loader:** Fabric

A lightweight utility mod that disables "Fair Play" restrictions in **Xaero's Minimap**. 

On many servers, administrators send packets to force-disable certain minimap features (like entity radar, cave maps, or entire map) to enforce their idea of "fair play". 
This mod bypasses those restrictions, giving you full control over your map functions regardless of server settings.

*Disclaimer: Use at your own risk. Some server rules may prohibit bypassing map restrictions.*
